- カテゴリは「基本」「句動詞」のみ
- 例文は社会人が使いやすい内容を優先
- 基本ではターゲット動詞だけを赤文字対象にする
- 句動詞では句動詞全体を赤文字対象にする
- 画像・目標日保存対策、下部バー余白修正を維持
